Roadmap of Future Development

Please feel free to use our Contact us page if you have ideas or concerns about our roadmap, and it will go directly to our devs!

High Priority

  1. Integrate current web apps into database systems
  2. Proper Log in systems with ease from Google
  3. Allow for students to find uploaded syllabi from instructors
  4. A place for students to view all of their syllabi from courses chosen
  5. Allow for instructors to manage their uploaded syllabi (edit/delete)
  6. Modify the site based on if a professor or student is viewing it
These will take precedence over other panels

Low Priority

  1. Multiple course syllabi in a single packet
  2. Fix site format issues with calendar page
  3. Allow for students to request a change from professors about a course
  4. Allow professors to accept/deny changes from requests from students
  5. Notify students of a change to a syllabi of their courses
  6. More tools to use when creating syllabi for a course
These will likely be pushed off unless the task is simple

Future Ideas/Tasks

  1. Allow for the site to be used with more schools
  2. Scan syllabi that has been uploaded and pull data from them
    based on common themes
  3. Add more customization from our calendars and syllabi for better
    student QoL
  4. More tools to make students organization clean and successful
  5. More tools to allow professors to easily convey
    information to their students
These will be looked into once happy with initial implementation
